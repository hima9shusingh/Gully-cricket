# 😂 Why I Built This?

Let's be honest...

Every gully cricket match ends like this...

> 🧑 "Bhai 42 banaye the!"

> 👦 "Nahi bhai 36 hi hue the!"

> 🧑 "Wide bhi add karo!"

> 👦 "Arre last ball count hi nahi hui!"

After hearing this argument almost every Sunday...

I decided...

**Fine... I'll build an app that never forgets the score. 😎🏏**
